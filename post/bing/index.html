接口介绍

本站提供了一个简单的必应每日一图的接口，

  • uri: https://bing.hxsup.top/bing/page
  • 参数：
    • page: 当前页码
    • page_size: 每一页的数量

type ImageInfo struct {
  Startdate     string `json:"startdate"`
  Enddate       string `json:"enddate"`

  Copyright     string `json:"copyright"`
